About Website#

Use hugo+PaperMod theme to create a website.

  1. install git, go, hugo
  2. hugo new site website
  3. cd website && git init
  4. git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
  5. add theme: ["PaperMod"] to hugo.yaml
  6. hugo new contents/posts/about_website.md
  7. change draft: true to draft: false in about_website.md
  8. hugo
  9. hugo server -D
  1. create a new repository on GitHub named user-name.github.io
  2. change baseURL: "https://user-name.github.io" in hugo.yaml
  3. cd public
  4. git init
  5. git add .
  6. git commit -m "first commit"
  7. git branch -M main
  8. git remote add origin https://github.com/user-name/user-name.github.io.git
  9. git push -u origin main
